# AI Гид — Умный ассистент путешественника (в разработке)

**Цель проекта** — создание офлайн-ассистента путешественника на основе RAG, который по названию города или региона выдаёт:

- Краткое описание  
- Главные достопримечательности  
- Практические советы (виза, транспорт, безопасность, деньги)  
- Ссылки на оригинальные статьи  

**Ключевая особенность** — **никаких галлюцинаций**: вся информация строго извлекается из текста русской Wikivoyage с помощью LLM (Mistral) с жёстким промптом.

---

### Статус проекта

| Компонент                    | Статус      | Комментарий                                   |
|------------------------------|-------------|-----------------------------------------------|
| Сбор данных (Wikivoyage)     | Готово      | 6919 статей, без редиректов                   |
| Структуризация через Mistral | Готово      | `wikivoyage_ru_final.jsonl` + CSV             |
| Telegram-бот                 | В разработке| Планируется на aiogram 3 + локальный поиск    |
| Веб-интерфейс                | В разработке| FastAPI + HTMX или Streamlit                  |
| Офлайн-поиск (RAG)           | Планируется | Llama.cpp / Ollama / vLLM                     |

**Сейчас доступен только полностью готовый и проверенный датасет.**
