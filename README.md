# AI Гид — Умный ассистент путешественника (в разработке)

**Цель проекта** — создание ассистента путешественника на основе RAG, который по названию города или региона выдаёт:

- Краткое описание  
- Главные достопримечательности  
- Практические советы (виза, транспорт, безопасность, деньги)  
- Ссылки на оригинальные статьи  

- **База знаний**: данные из русской Wikivoyage хранятся в Elasticsearch (развернут через Docker на удаленном сервере, где хостится сайт)
- **Процесс работы**:
  1. Запрос пользователя эмбеддится
  2. Поиск релевантной информации в RAG (Elasticsearch с данными из Wikivoyage)
  3. Если информации недостаточно для ответа, используется tool для поиска дополнительной информации в интернете
